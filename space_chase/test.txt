Directions to test Space Chase

Ensure Python and Pygame are installed (pip install pygame).

Navigate to project directory.
Run the game script:
python space_chase.py

Gameplay Testing:
Verify spaceship responds correctly to arrow keys or WASD keys
Confirm the game ends on collision with an enemy (future feature)
Confirm the game ends on collision with an enemy projectile (future feature)
Make sure that when you collect a credit the number of credits increases in the top left of the screen (future feature)
Ensure that when you collide with the rocketship it takes you to the next level (future feature)
